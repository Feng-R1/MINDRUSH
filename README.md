**MindRush** 是一款基于 Rust 语言开发的开源 AI 翻译桌面应用，采用 Tauri 2 框架构建（后端 Rust + 前端 React/TypeScript）。
核心优势在于：支持 EPUB、PDF、Markdown 三种文档格式的**整书翻译**，输入输出均为**文件**，且通过占位符标记技术（如 `\x00EPUB_N\x00`）和 lol_html / pulldown-cmark 等解析器实现**高保真格式保留**（排版、标题、代码块、图片等均不丢失）。
可翻译**上千页**的书籍，采用**智能分批处理**（每批最多100段，超限自动拆分）和**指数退避重试**机制，集成 OpenAI / Anthropic / 本地 LLM 等多种翻译引擎，支持术语表、翻译缓存、进度追踪等功能。
